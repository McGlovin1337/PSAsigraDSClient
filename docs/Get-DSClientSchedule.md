# Get-DSClientSchedule

## SYNOPSIS
Display the Schedules configured on the DS-Client computer

## SYNTAX

```
Get-DSClientSchedule [<CommonParameters>]
```

## DESCRIPTION
Display the Schedules configured on the DS-Client computer

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-DSClientSchedule
```

Lists the DS-Client Schedules
