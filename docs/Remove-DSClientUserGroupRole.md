# Remove-DSClientUserGroupRole

## SYNOPSIS
Remove a Role from the DS-Client

## SYNTAX

```
Remove-DSClientUserGroupRole [-RoleId] <Int32> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Remove a User or Group Role from the DS-Client

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-DSClientUserGroupRole -RoleId 5
```

Remove the DS-Client User or Group Role with RoleId 5
