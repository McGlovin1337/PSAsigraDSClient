# Remove-DSClientArchiveFilterRule

## SYNOPSIS
Remove an Archive Filter Rule from DS-Client

## SYNTAX

```
Remove-DSClientArchiveFilterRule [-Name] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Remove and Archive Filter Rule and all associated Filters from the DS-Client

## EXAMPLES

### Example 1
```powershell
PS C:\> Remove-DSClientArchiveFilterRule -Name 'MyFilterRule'
```

Removes the Archive Filter Rule named 'MyFilterRule' from the DS-Client
