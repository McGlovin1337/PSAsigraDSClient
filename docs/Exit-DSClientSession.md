# Exit-DSClientSession

## SYNOPSIS
Closes the DS-Client connection

## SYNTAX

```
Exit-DSClientSession [<CommonParameters>]
```

## DESCRIPTION
Closes the DS-Client connection and destroys the session.

## EXAMPLES

### Example 1
```
PS C:\> Exit-DSClientSession
```

Closes the DS-Client connection and destroys the session.
